## CEO Monthly Review Checklist

Run this every 1st of the month:

**Financial:**
- [ ] MRR vs last month (+/-)
- [ ] Burn rate vs budget
- [ ] Runway remaining (months)
- [ ] New clients this month
- [ ] Churn this month

**Product:**
- [ ] Features shipped
- [ ] Bugs critical/unresolved
- [ ] NPS score from beta clients
- [ ] Inspector violations count

**Team:**
- [ ] Everyone paid on time
- [ ] Any resignations / hiring needs
- [ ] Team morale (1–10 rating from CEO gut)
- [ ] Training/growth happening

**Legal/Ops:**
- [ ] Compliance issues raised
- [ ] Any client complaints
- [ ] Documents/registrations expiring
